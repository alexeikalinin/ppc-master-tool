# Claude Skills System — Portable Template

Самообучающаяся система скиллов и агентов для Claude Code.
Скопируй эту папку в новый проект и следуй инструкции ниже.

---

## Установка в новый проект

### Шаг 1 — Скопируй папку

Скопируй содержимое этой папки в корень нового проекта:

```
новый-проект/
├── .claude/
│   └── skills/           ← скопировать сюда содержимое папки skills/
├── CLAUDE.md             ← добавить сниппет из CLAUDE-SNIPPET.md
└── ...
```

То есть:
- папку `skills/` → в `.claude/skills/`
- из `CLAUDE-SNIPPET.md` → скопировать текст в `CLAUDE.md` проекта

### Шаг 2 — Добавь в CLAUDE.md

Открой (или создай) `CLAUDE.md` в корне нового проекта и вставь текст из `CLAUDE-SNIPPET.md`.

### Шаг 3 — Настрой предпочтения

Открой `.claude/skills/knowledge-base/assets/preferences.json` и заполни под свой проект:
- язык вывода (output)
- стиль общения (verbosity)

### Шаг 4 — Всё готово

Запусти Claude Code в новом проекте. Система начнёт работать автоматически:
- читать уроки перед задачами
- записывать фидбэк
- логировать изменения
- предлагать создавать новые скиллы при повторяющихся задачах

---

## Что делает система

```
Запрос → Планировщик → Исполнитель → Ревьюер → База знаний
                                           ↓
                              Уроки → следующий запрос умнее
```

### Скиллы

| Скилл | Назначение | Когда срабатывает |
|---|---|---|
| `meta-orchestrator` | Мозг системы, маршрутизация задач | Любая сложная или многоходовая задача |
| `skill-factory` | Создаёт новые скиллы | Когда задача повторяется 3+ раза |
| `quality-loop` | Учится на фидбэке | Когда ты говоришь "не так", "отлично", "запомни" |
| `knowledge-base` | Долгосрочная память | Фоново — читается и пишется другими скиллами |
| `change-logger` | Логирует изменения кода | В конце сессии с изменениями кода |

## Добавление новых скиллов

Создай папку `skills/my-skill/SKILL.md` со структурой:

```markdown
---
name: my-skill
description: "Что делает + когда срабатывает"
---

# My Skill

## Overview
...

## Step-by-step
1. ...
2. ...

## Anti-patterns
- Не делай X
```

Затем добавь запись в `knowledge-base/assets/skills-registry.json`.
